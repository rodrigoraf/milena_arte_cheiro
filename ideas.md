# Brainstorm de Design - Milena Arte e Cheiro

## Abordagem 1: Minimalismo Botânico Contemporâneo
**Design Movement:** Minimalism + Botanical Illustration  
**Probabilidade:** 0.08

### Core Principles
- Simplicidade elegante com foco em tipografia clara
- Espaço negativo generoso para respiração visual
- Elementos florais como acentos, não como decoração excessiva
- Paleta reduzida e intencional

### Color Philosophy
- **Branco puro** (fundo principal) - pureza e artesanalidade
- **Rosa pálido** (#F5E6E0) - delicadeza e feminilidade suave
- **Verde sálvia** (#8B9A7F) - conexão com natureza, sofisticação
- **Marrom quente** (#A67C52) - terra, autenticidade artesanal
- Uso mínimo de cores - máximo impacto

### Layout Paradigm
- Grid assimétrico com coluna principal larga e lateral vazia
- Tipografia como protagonista
- Imagens grandes e respiradas
- Navegação horizontal simples no topo

### Signature Elements
- Linhas florais delicadas como divisores
- Ícones minimalistas de flores em traço fino
- Tipografia serif elegante para títulos

### Interaction Philosophy
- Transições suaves e sutis
- Hover effects discretos (mudança de cor leve)
- Scroll revelando conteúdo gradualmente

### Animation
- Fade-in suave ao carregar seções
- Hover: mudança de opacidade de 100% para 95%
- Micro-interações: ícones com transição de cor em 300ms

### Typography System
- **Display:** Playfair Display (serif elegante, títulos)
- **Body:** Lato (sans-serif limpo, legibilidade)
- Hierarquia clara: 48px → 32px → 18px → 14px

## Abordagem 2: Romantismo Artesanal com Texturas
**Design Movement:** Romantic + Handmade Aesthetic  
**Probabilidade:** 0.07

### Core Principles
- Texturas e padrões que remetem ao processo artesanal
- Cores quentes e acolhedoras
- Sensação de autenticidade e imperfeição controlada
- Elementos decorativos que contam uma história

### Color Philosophy
- **Rosa coral** (#E8B4A8) - caloroso e convidativo
- **Branco creme** (#FFFAF0) - suavidade natural
- **Verde musgo** (#7A9B6F) - orgânico e vivo
- **Marrom chocolate** (#6B4423) - profundidade e terra
- Tons secundários: ouro suave (#D4AF9F) para acentos

### Layout Paradigm
- Seções com fundo alternado (texturizado vs. limpo)
- Cards com sombras suaves e bordas arredondadas
- Padrão floral sutil como fundo em seções específicas
- Imagens com molduras decorativas

### Signature Elements
- Padrão floral repetido como watermark
- Bordas decorativas em seções
- Flores prensadas como elementos visuais

### Interaction Philosophy
- Animações que simulam movimento natural (flores balançando)
- Hover com elevação (shadow aumenta)
- Cliques com feedback visual suave

### Animation
- Entrada de elementos com bounce suave
- Hover: scale 1.02 com shadow expansion
- Flores animadas em background (movimento leve contínuo)

### Typography System
- **Display:** Cormorant Garamond (serif luxuoso, títulos)
- **Body:** Montserrat (sans-serif moderno, legibilidade)
- Pesos: 700 para títulos, 400-500 para corpo

## Abordagem 3: Modernismo Orgânico com Gradientes Suaves
**Design Movement:** Organic Modernism + Soft Gradients  
**Probabilidade:** 0.09

### Core Principles
- Formas orgânicas e curvas naturais
- Gradientes suaves entre cores complementares
- Equilíbrio entre modernidade e natureza
- Movimento visual através de composição

### Color Philosophy
- **Gradiente principal:** Rosa (#F5D5E0) → Branco (#FFFFFF)
- **Gradiente secundário:** Verde claro (#D4E8D4) → Branco
- **Acentos:** Marrom dourado (#B8956A) para CTAs
- Uso de gradientes radiais e lineares para profundidade

### Layout Paradigm
- Seções com formas orgânicas (wavy dividers, blob shapes)
- Imagens em containers com bordas arredondadas assimétricas
- Layout fluido com elementos que se sobrepõem
- Navegação flutuante ou sticky com design moderno

### Signature Elements
- Formas blob em cores suaves como backgrounds
- Wavy dividers entre seções
- Flores em ilustração moderna (não fotográficas)

### Interaction Philosophy
- Micro-interações com movimento fluido
- Parallax suave ao scroll
- Elementos que reagem ao movimento do mouse

### Animation
- Entrada com slide + fade combinados
- Hover: rotação suave (2-3 graus) + mudança de cor
- Scroll: parallax em imagens (movimento 20-30% mais lento)

### Typography System
- **Display:** Poppins Bold (moderno, geométrico)
- **Body:** Poppins Regular (coesão visual)
- Hierarquia: 56px → 36px → 20px → 14px

---

## Decisão Final
**Abordagem Selecionada: Minimalismo Botânico Contemporâneo**

Esta abordagem foi escolhida porque:
1. Transmite elegância e profissionalismo
2. Deixa o foco nos produtos (sabões e velas)
3. Reflete a qualidade artesanal sem parecer amador
4. Escalável e versátil para futuras expansões
5. Carrega a identidade floral solicitada de forma sofisticada
